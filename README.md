# React Native Mobile App

Applicazione mobile cross-platform sviluppata in React Native per il corso di Mobile Computing presso l’Università degli Studi di Milano.  

Il progetto implementa un sistema di food delivery basato su consegne rapide tramite drone, interfacciandosi con API REST fornite dal docente. :contentReference[oaicite:8]{index=8}

---

## Tecnologie utilizzate

- React Native
- JavaScript
- Expo / React Navigation
- REST API
- AsyncStorage
- Google Maps
- JSON

---

## Funzionalità implementate

- Session Management
- Profilo utente (Gestione dei dati personali e dei dati di pagamento)
- Menu List & Details (Visualizzazione dei menù disponibili con i relativi dettagli)
- Delivery Tracking
- State Persistence (sessione utente, ultima schermata visualizzata, dati locali necessari)

---

## Competenze sviluppate

Questo progetto mi ha permesso di approfondire:
- sviluppo mobile cross-platform
- gestione API REST
- state management
- persistenza locale
- validazione form
- navigazione mobile
- gestione immagini e dati remoti
- progettazione UI/UX mobile
